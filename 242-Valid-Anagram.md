## Valid Anagram
给定两个字符串 s 和 t, 写一个确定 t 是否是 s 的异构词对函数

#### 示例 1:
```
输入: s = "anagram", t = "nagaram"
输出: true
```
#### 示例 2:
```
输入: s = "rat", t = "car"
输出: false
```
#### 注意
你可以假定字符串只包含小写字母

#### 后续
如果输入包含 unicode 字符呢? 对于这种情况你将如何调整你的解法
