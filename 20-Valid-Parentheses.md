## Valid Parentheses
给定一个只包含字符 ``'(', ')', '[', ']', '{', '}'``, 确定输入的字符串时候有效  
如果一个输入的字符串是有效的:
- 左括号必须匹配相同类型的右括号
- 左括号必须匹配正确顺序的右括号

注意, 空字符串也被认为是有效的

#### 示例 1：
```
输入: "()"
输出: true
```

#### 示例 2:
```
输入: "()[]{}"
输出: true
```

#### 示例 3:
```
输入: "(]"
输出: false
```

#### 示例 4:
```
输入: "([)]"
输出: false
```

#### 示例 5:
```
输入: "{[]}"
输出: true
```
