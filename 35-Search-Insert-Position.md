## Search Insert Position
给定一个有序的数组和一个目标值, 如果发现目标值则返回其索引, 如果不是, 则返回按顺序插入的索引; 你可以假定数组中没有重复值

#### 示例 1:
```
输入: [1,3,5,6], 5
输出: 2
```

#### 示例 2:
```
输入: [1,3,5,6], 2
输出: 1
```

#### 示例 3:
```
输入: [1,3,5,6], 7
输出: 4
```

#### 示例 4:
```
输入: [1,3,5,6], 0
输出: 0
```
