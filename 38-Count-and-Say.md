## Count and Say
count-and-say 序列是一个整数序列, 前五项如下
```
1.     1
2.     11
3.     21
4.     1211
5.     111221
```
1 读作 "一个 1" 或者 11  
11 读作 "两个 1" 或者 21  
21 读作 "一个 2, 然后一个 1", 或者 1211  
给定一个整数 n, 生成第 n 项 count-and-say 序列  
注意: 每项的整数序列将会被表示为字符串

#### 示例 1:
```
输入: 1
输出: "1"
```

#### 示例 2:
```
输入: 4
输出: "1211"
```
